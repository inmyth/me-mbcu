## ssl certificate 

When renewing certificate, the fastest way to confirm is having it send an email to admin@mbcu.me . 
This requires an email address which can be set up with GoDaddy's Microsoft 365. 
If using email, let GoDaddy set up DNS table. Many things currently there are related to email.
